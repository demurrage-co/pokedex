This is public facing repository for the Pokedex project. This is the worlds most accurate carrier lookup tool for phone number. This tool is made for scambaiters in mind to report scams to legitimate services they are abusing to take them down. This tool is currently in it's alpha testing phase. It will be released to the public within Q1 of 2026. Do note that pokedex is not the final release name, it is an internal name used to refer to this project. This repository will be renamed once we have a name and a logo.
